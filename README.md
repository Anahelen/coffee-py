# Café ☕️ con sabor a Python✨

Mi primer proyecto en Python. Es un pequeño sistema para manejar pedidos de café, menú y historial.  

## Cómo usarlo
1. Clona el repositorio
2. Ejecuta `python main.py`

## Archivos
- `main.py` → archivo principal
- `menu.py` → menú de productos
- `pedidos.py` → gestión de pedidos
- `historial.py` → historial de pedidos
